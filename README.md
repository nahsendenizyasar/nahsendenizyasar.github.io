# Nahsen Deniz Yaşar – Portfolio Website

This is a static personal portfolio website for GitHub Pages.

## Files

- `index.html` – main website file
- `assets/profile.jpg` – profile image used in the hero section

## Publish on GitHub Pages

1. Create a GitHub account or log in.
2. Create a new repository.
3. For a personal page, name it exactly:
   `YOUR-GITHUB-USERNAME.github.io`
4. Upload `index.html` and the full `assets` folder.
5. Commit the files.
6. Open **Settings → Pages**.
7. Set source to **Deploy from a branch**.
8. Select branch **main** and folder **/root**.
9. Save.
10. Your website will be available at:
   `https://YOUR-GITHUB-USERNAME.github.io`

## Language switch

The site contains German, English and Turkish text. German is the default language.
